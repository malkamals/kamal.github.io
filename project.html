My Game Project:

Hangman

  • Apa itu Hangman?
    • Hangman adalah permainan menebak kertas dan pensil untuk dua atau lebih pemain. Namun, seiring berkembangnya teknologi, Hangman bisa dimainkan secara digital. Pada game yang saya buat bisa dimainkan secara single player. Pembuatan game tersebut menggunakan Cocos2d-x. Saya membuatkan 10 kategori dan 5 scene dalam setiap kategori, totalnya terdapat 55 scene. Game ini dibuat saat Magang di Educa Studio.

    QuestZo

  • Apa itu QuestZo?
    • QuestZo adalah game edukasi berbasis teka - teki yang saya buat bersama teman saya Fadhil Himawan. Dalam game ini, terdapat 6 kategori dan 10 scene dalam setiap kategori. Game ini dibuat dengan software Construct 2 untuk memudahkan proses development. Game ini ditujukan untuk siswa kelas 1 sekolah dasar. Game ini masih berada di tahap pengembangan, dan akan selesai pada bulan Januari 2021.
